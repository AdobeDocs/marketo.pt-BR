# Meu token: Campanha CRM

<br> 

Veja como configurar um token de campanha do CRM para uso em seus programas e pastas de campanha.

>[!NOTE]
>
>Meus tokens da campanha do CRM só estão disponíveis para integrações do Salesforce.

1. Na guia Meus tokens do seu programa ou pasta, arraste um token **[!UICONTROL Campanha CRM]** para a tela Tokens locais.

   ![Imagem Um](/help/sky/assets/my-tokens/my-token-crm-campaign/my-token-crm-campaign-1.png)

2. Insira um nome para o novo token e selecione uma campanha CRM no menu suspenso. Clique em **[!UICONTROL Salvar]**.

   ![Imagem dois](/help/sky/assets/my-tokens/my-token-crm-campaign/my-token-crm-campaign-2.png)

>[!NOTE]
>
>Este Meu token só estará disponível se sua assinatura tiver uma integração CRM.
