# Meu token: Texto formatado

<br> 

Veja como configurar um token de Rich Text para ser usado em seus programas e pastas de campanha.

1. Na guia [!UICONTROL Meus tokens] do seu programa ou pasta, arraste um token **Rich Text** para a tela [!UICONTROL Tokens locais].

   ![Imagem Um](/help/sky/assets/my-tokens/my-token-rich-text/my-token-rich-text-1.png)

1. Insira um nome para o novo token e adicione o texto que desejar no editor. Clique em **[!UICONTROL Salvar]**.

   ![Imagem dois](/help/sky/assets/my-tokens/my-token-rich-text/my-token-rich-text-2.png)
