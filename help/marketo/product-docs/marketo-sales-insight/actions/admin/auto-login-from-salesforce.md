# Login Automático de [!DNL Salesforce] {#auto-login-from-salesforce}

O logon automático permite que [!DNL Sales Insight Actions] usuários se autentiquem automaticamente em [!DNL Sales Insight Actions] ao trabalhar em [!DNL Salesforce].

Para usar o login automático:

* [Configurações de Gerenciamento de Logon](/help/marketo/product-docs/marketo-sales-insight/actions/admin/login-management-settings.md) devem ser definidas como &quot;[!UICONTROL Somente Salesforce]&quot;

* O usuário deve ter sido convidado para usar [!DNL Sales Insight Actions]

* O usuário deve ter ativado a conta conectando a conta [!DNL Salesforce] à instância [!DNL Sales Insight Actions]

* O nome de usuário [!DNL Sales Insight Actions] do usuário deve corresponder ao nome de usuário [!DNL Salesforce] (ou o nome de usuário e email [!DNL Salesforce] são correspondentes)
