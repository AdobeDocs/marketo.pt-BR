# Navegação na página inicial do Analytics {#navigating-the-analytics-home-page}

1. Vá para a área **[!UICONTROL Analytics]**.

   ![](assets/image2015-4-27-8-3a38-3a10.png)

1. Selecione um [tipo de relatório](/help/marketo/product-docs/reporting/basic-reporting/report-types/report-type-overview.md).

   ![](assets/image2015-4-27-8-3a38-3a22.png)

1. Depois de executar o relatório, clique no espaço de trabalho para voltar para a **Página inicial do Analytics**.

   ![](assets/image2015-4-27-8-3a38-3a34.png)

   Ótimo! Você sabe como navegar pela página inicial do Analytics!
