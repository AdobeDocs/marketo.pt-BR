# Requisitos mínimos do sistema {#minimum-system-requirements}

Se você estiver se perguntando o que precisa fazer logon e usar o Marketo, veja as informações que está procurando.

>[!NOTE]
>
>iOS e Android ainda não são compatíveis.

* Computador ou laptop com Windows, OS X ou Linux
* Teclado e mouse
* Monitor com resolução mínima de 1280 x 768
* Navegador atualizado, consulte [Navegadores compatíveis](/help/marketo/product-docs/administration/setup-administration/supported-browsers.md){target=&quot;_blank&quot;} para obter detalhes
* Conexão com a Internet

>[!NOTE]
>
>Se sua equipe de TI restringir o acesso à Web usando uma  lista de permissões, peça para adicioná-la `*.marketo.com` para permitir todos os recursos e websockets do Marketo.
