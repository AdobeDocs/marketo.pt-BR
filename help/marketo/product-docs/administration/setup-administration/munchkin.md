# [!DNL Munchkin] {#munchkin}

[!DNL Munchkin] é o trecho JavaScript que você coloca no seu site. Ele rastreia visitas, cliques e preenchimentos de formulário. Ele também tem uma API avançada para fazer todo tipo de coisas legais.

Confira o [página do desenvolvedor em [!DNL Munchkin]](https://developers.marketo.com/documentation/websites/lead-tracking-munchkin-js/){target="_blank"} para saber mais.
