# Criar tags personalizadas {#create-custom-tags}

As tags ajudarão você a organizar seus programas, enquanto os canais ajudarão na coleta de dados para fins de relatório.

>[!NOTE]
>
>Confira [Entendendo tags](/help/marketo/product-docs/core-marketo-concepts/programs/working-with-programs/understanding-tags.md) para obter mais informações.

>[!NOTE]
>
>**Permissões de administrador necessárias**

## Criar um novo tipo de tag {#create-a-new-tag-type}

Para criar um novo tipo de tag:

1. Vá para a seção **Admin**.

   ![](assets/image2015-4-23-14-3a37-3a48.png)

1. Clique em **Tags**.

   ![](assets/image2015-4-23-14-3a41-3a18.png)

1. Clique em **Novo** e selecione **Novo Tipo de Tag.**

   ![](assets/image2015-4-23-14-3a42-3a45.png)

1. Insira um nome para o **Tipo de tag**.

   ![](assets/image2015-4-23-14-3a48-3a58.png)

1. Insira o **Valor** desejado para o Tipo de tag. Clique em **Add Other** para inserir valores de tag adicionais.

   ![](assets/image2015-4-22-11-3a30-3a30.png)

1. Selecione a quais tipos de Programas esta tag se aplica.

   ![](assets/image2015-4-22-11-3a39-3a7.png)

   >[!NOTE]
   >
   >Selecione **Obrigatório para todos os programas** para adicionar esta tag sempre que um novo programa for criado.

1. Clique em **Criar** para salvar.

   ![](assets/image2015-4-22-11-3a38-3a34.png)

   Pronto! Agora você pode criar tags personalizadas para seus programas.
