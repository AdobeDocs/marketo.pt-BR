# Incluir um arquivo ICS de evento de calendário em uma página de aterrissagem {#include-a-calendar-event-ics-file-in-a-landing-page}

A **Arquivo de calendário** O token permite adicionar um link de evento de calendário (.ics) às suas páginas de aterrissagem do Marketo.

>[!PREREQUISITES]
>
>* [Criar um arquivo de evento de calendário (.ics)](/help/marketo/product-docs/email-marketing/general/functions-in-the-editor/create-a-calendar-event-ics-file.md)


1. No editor de landing page, clique em **{...}** para inserir um token.

   ![](assets/image2015-7-8-17-3a51-3a29.png)

1. Selecione o **Arquivo de calendário** token e clique **Inserir**.

   >[!CAUTION]
   >
   >Os seguintes tokens não são compatíveis com landing pages:
   >
   >* URL de membro.webinar


   ![](assets/image2015-1-6-16-3a31-3a28.png)

1. Clique em **Salvar**.

   As pessoas verão uma página de aterrissagem semelhante a esta:

   ![](assets/image2015-1-6-16-3a42-3a51.png)

Doce! Tudo deveria estar funcionando bem agora. Certifique-se de testar.
