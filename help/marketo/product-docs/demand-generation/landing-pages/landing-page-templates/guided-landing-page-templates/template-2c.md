# Modelo 2C {#template-2c}

Clique com o botão direito do mouse para [baixar o Modelo 2C](https://experienceleague.adobe.com/landing/marketo/lp-templates/template-2c.html)

![](assets/image2015-6-4-9-3a31-3a46.png)

Esse template inclui o seguinte conteúdo:

* Um cabeçalho com logotipo e botão (opcional)
* Uma seção principal

   * inclui imagem de fundo herói, cabeçalho, slogan, lista com marcadores e vídeo.

* Uma seção do corpo com texto e formulário (opcional)
* Rodapé (opcional)
