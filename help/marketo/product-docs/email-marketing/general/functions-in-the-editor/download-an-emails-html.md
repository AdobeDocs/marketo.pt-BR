# Baixar um HTML de email {#download-an-emails-html}

Para backups e outros propósitos, a Marketo permite baixar o conteúdo do HTML de um email.

1. Encontre o email e selecione-o. No **Ações de email** clique suspenso **Baixar o HTML**.

   ![](assets/one-4.png)

É tão fácil!
