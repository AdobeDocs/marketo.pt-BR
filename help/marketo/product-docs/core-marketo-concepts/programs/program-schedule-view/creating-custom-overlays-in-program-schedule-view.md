# Criando sobreposições personalizadas na exibição de programação {#creating-custom-overlays-in-program-schedule-view}

É possível criar sobreposições personalizadas para exibir entradas relevantes às suas necessidades.

1. Clique no menu suspenso **Agenda**.

   ![](assets/image2014-9-24-10-3a20-3a11.png)

1. Selecione **Sobreposições**.

   ![](assets/image2014-9-24-10-3a20-3a17.png)

1. Selecione os tipos de entrada que você deseja que estejam visíveis em sua sobreposição.

   ![](assets/image2014-9-24-10-3a20-3a26.png)

1. Você também pode filtrar por [tags de programa](/help/marketo/product-docs/core-marketo-concepts/programs/working-with-programs/understanding-tags/use-tags-in-a-program.md).

   ![](assets/image2014-9-24-10-3a20-3a32.png)

   Ótimo. Agora, sua sobreposição exibirá somente as entradas definidas.

   ![](assets/image2014-9-24-10-3a20-3a37.png)
