## Regras de conflito {#conflict-rules}

* **Marcas de programa**
   * Criar tags nesta assinatura (Padrão) - _Recomendado_
   * Ignorar

* **Modelo de landing page com o mesmo nome**
   * Copiar modelo original (padrão) - _Recomendado_
   * Usar modelo de destino

* **Imagens com o mesmo nome**
   * Manter ambos os arquivos (Padrão) - _Recomendado_
   * Substituir item desta inscrição

* **Modelos de e-mail com o mesmo nome**
   * Manter ambos os modelos (Padrão) - _Recomendado_
   * Substituir modelo existente

CAPTURA DE TELA DAS REGRAS DE CONFLITO

>[!TIP]
>
>Para utilizar o programa como está, mantenha as regras de conflito padrão selecionadas.

## Práticas recomendadas {#best-practices}

* Considere adicionar outras Campanhas inteligentes para atender a cada uma das necessidades de status do ciclo de vida que você possa estar rastreando em sua organização. Cada campanha integrada neste programa deve ser um exemplo da criação de práticas recomendadas e não específica para todos os casos de uso. Lembre-se de atualizar as Campanhas inteligentes para lidar com o processo específico de gerenciamento do ciclo de vida da pessoa.
