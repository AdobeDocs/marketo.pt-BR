# Criar um {#creating-a-favorite} favorito

Marque um cartão como favorito para facilitar a referência futura.

1. Abra o menu Cartão.

   ![](assets/image2015-7-14-16-3a28-3a54.png)

1. Toque em **Favorito**.

   ![](assets/image2015-7-14-16-3a36-3a22.png)

   Isso é tudo!
