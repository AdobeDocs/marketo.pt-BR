# Criar um favorito {#creating-a-favorite}

Marque um cartão como favorito para uma referência futura fácil.

1. Abra o menu do cartão.

   ![](assets/image2015-7-14-16-3a28-3a54.png)

1. Toque em **Favorito**.

   ![](assets/image2015-7-14-16-3a36-3a22.png)

   Isso é tudo que há nisso!
