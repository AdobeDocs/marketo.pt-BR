# Usar Listas integradas/inteligentes do sistema {#use-built-in-system-smart-lists}

Marketo tem algumas listas inteligentes incorporadas muito úteis. Vamos dar uma olhada!

>[!NOTE]
>
>O Marketo está mudando termos como Blacklist e Whitelist para Lista de bloqueios e Lista de permissões em nosso produto. Durante esta atualização, você pode ver os termos antigos em nossa interface do usuário e nas capturas de tela da documentação, bem como os novos termos em nosso texto de documentação. Pedimos desculpas por qualquer confusão.

1. Vá para o banco de dados***.

   ![](assets/db.png)

1. Abra a pasta Listas **inteligentes do** sistema para revelar a coleção.

   ![](assets/two.png)

1. Para ver um exemplo: Selecione **Todas as pessoas** e clique na guia **Pessoas** .

   ![](assets/three.png)

   >[!NOTE]
   >
   >Os filtros nas listas inteligentes do sistema não precisam ser aplicados a cada lista/campanha inteligente. O conteúdo é reconhecido automaticamente.

   | Nome da lista | Descrição |
   |---|---|
   | Todas as pessoas | Todas as pessoas no seu banco de dados do Marketo |
   | Pessoas sem Assinatura | Estas pessoas só podem receber e-mails operacionais; isso é geralmente controlado pela própria pessoa. |
   | Suspenso de marketing | Estas pessoas só podem receber e-mails operacionais; isso é geralmente controlado por você, o comerciante. |
   | lista de bloqueios | Essas pessoas não receberão nenhum email. |
   | Endereços de email enviados | Pessoas com um endereço de email não entregue ou que rejeitam seu email. |
   | Possíveis Duplicados | Pessoas que podem ser duplicadas no banco de dados do Marketo. |

>[!NOTE]
>
>Listas inteligentes do sistema não podem ser excluídas. Com exceção da lista Possíveis Duplicados, eles também não podem ser editados.
