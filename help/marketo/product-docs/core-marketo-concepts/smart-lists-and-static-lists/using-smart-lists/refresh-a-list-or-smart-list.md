# Atualizar uma lista ou lista inteligente {#refresh-a-list-or-smart-list}

Se você tiver executado uma Smart List e alguns minutos tiverem passado, os resultados poderão ser diferentes agora - atualize para descobrir.

## Atualizar resultados {#refresh-results}

1. Para atualizar dados na guia **[!UICONTROL Pessoas]** de uma Smart List, clique no ícone de atualização.

   ![](assets/refresh-a-list-or-smart-list-1.png)

1. A Smart List é executada novamente e mostra um conjunto de resultados mais atualizado.

   ![](assets/refresh-a-list-or-smart-list-2.png)

>[!TIP]
>
>Às vezes, ao executar uma Smart List e voltar posteriormente, você poderá ver a palavra &quot;Sobre&quot; na frente da contagem de pessoas no canto inferior direito. Isso indica que o número é aproximado - clique na própria contagem para atualizá-la e obter uma contagem atualizada e precisa.
