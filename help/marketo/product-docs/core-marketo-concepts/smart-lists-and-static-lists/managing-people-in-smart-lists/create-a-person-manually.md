# Criar uma pessoa manualmente {#create-a-person-manually}

Há muitas maneiras de colocar uma pessoa no Marketo. Se precisar criar um manualmente, veja como fazê-lo.

>[!CAUTION]
>
>A Marketo faz **not** são compatíveis com endereços de email que contêm emojis.

1. Vá para o **Banco de dados**.

   ![](assets/db-1.png)

1. Em **Novo**, clique em **Nova pessoa**.

   ![](assets/two-2.png)

1. Insira as informações da pessoa e clique em **Criar**.

   ![](assets/three-2.png)

Essa é uma maneira rápida e fácil de adicionar uma única pessoa no Marketo!
