# Alterar proprietário na Microsoft {#change-owner-in-microsoft}

Se você tiver pessoas já atribuídas a um proprietário, poderá usar essa etapa de fluxo para atribuí-las novamente a outro proprietário.

>[!NOTE]
>
>Esta etapa de fluxo **só funcionará quando for usada com acionadores**, não filtros, na sua campanha inteligente.

![](assets/one-1.png)

**Uso**

1. Basta escolher o proprietário para o qual você deseja mudar e ir!

   ![](assets/two-1.png)

   >[!NOTE]
   >
   >Se o registro ainda não existir em sua conta do Dynamics, ele será sincronizado e atribuído ao usuário selecionado.
