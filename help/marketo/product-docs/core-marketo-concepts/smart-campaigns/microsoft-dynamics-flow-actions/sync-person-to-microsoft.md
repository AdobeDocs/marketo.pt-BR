# Sincronizar Pessoa com o Microsoft {#sync-person-to-microsoft}

Este recurso é somente para aqueles que integram o Marketo ao Microsoft Dynamics.

## Visão geral {#overview}

Esta etapa de fluxo inserirá pessoas criadas pelo Marketing no seu Dynamics CRM.

![](assets/one.png)

## Uso {#usage}

Você pode definir um usuário do Dynamics como proprietário da pessoa.

![](assets/two.png)

>[!NOTE]
>
>Ao usar a ação de fluxo &quot;Sincronizar Pessoa com a Microsoft&quot; (somente em uma campanha de disparo), o cliente potencial/contato será criado em tempo real no Dynamics.
