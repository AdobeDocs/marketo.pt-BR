# Exportar dados de campanha da Web {#export-web-campaign-data}

Siga estas etapas simples para exportar seus dados de campanha da Web.

1. Ir para **Campanhas da Web**.

   ![](assets/one-2.png)

1. Na parte superior direita da página, clique no ícone Exportar CSV .

   ![](assets/two-2.png)

1. Abra ou salve o arquivo.

   ![](assets/three-2.png)

1. Visualize seu arquivo para ver as estatísticas úteis.

   ![](assets/four-1.png)
