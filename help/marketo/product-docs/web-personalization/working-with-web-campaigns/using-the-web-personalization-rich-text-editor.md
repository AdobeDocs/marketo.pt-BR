# Uso do Editor de Rich Text de Personalização da Web {#using-the-web-personalization-rich-text-editor}

![](assets/one.png)

O editor de rich text de Personalização da Web encontrado na página Definir campanhas permite a formatação, vinculação e inserção de imagens do texto. Ele inclui opções de formatação semelhantes aos processadores de texto padrão.
