# Usando o Editor de Rich Text de Personalização da Web {#using-the-web-personalization-rich-text-editor}

![](assets/one.png)

O editor Rich Text de Personalização da Web encontrado na página Definir Campanhas permite a formatação de texto, vinculação e inserção de imagem. Inclui opções de formatação semelhantes aos processadores de texto padrão.

>[!NOTE]
>
>Os Tokens Web usam [dados de pessoas](http://docs.marketo.com/display/DOCS/Manage+Person+Data) do seu Banco de Dados de Marketing que é gerenciado na Personalização Web em Configurações de Conta > Banco de Dados. Se o token da Web for um novo campo de banco de dados que ainda não esteja em Configurações de conta > Banco de dados, ele será automaticamente adicionado e poderá levar até 24 horas para ser ativado.
