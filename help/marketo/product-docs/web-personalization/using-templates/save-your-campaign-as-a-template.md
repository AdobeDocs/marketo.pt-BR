# Salvar a campanha como um modelo {#save-your-campaign-as-a-template}

Você já passou algum tempo criando a campanha perfeita para a web? Agora você pode salvá-lo como um modelo para reutilização fácil no futuro.

1. Vá para **[!UICONTROL Campanhas da Web]**.

   ![](assets/web-campaigns-hand-1.jpg)

1. Procure a campanha que deseja salvar como um template.

   ![](assets/search-for-campaign.jpg)

1. Clique no ícone Edit.

   ![](assets/my-campaign-edit.jpg)

1. Marque **[!UICONTROL Usar como modelo]** e clique em **[!UICONTROL Salvar]**.

   ![](assets/image2015-2-25-19-3a56-3a58.png)   ![](assets/image2015-2-25-19-3a56-3a37.png)

1. Pronto! Na próxima vez que criar uma campanha e selecionar um modelo, marque [!UICONTROL Meus Modelos] na página Definir Campanhas para ver os modelos que você salvou.

   ![](assets/image2014-9-17-20-3a55-3a31.png)
