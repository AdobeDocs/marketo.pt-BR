# Habilitar/desabilitar assinaturas de relatório {#enable-disable-report-subscriptions}

O [!UICONTROL Web Personalization] tem vários relatórios úteis enviados por email. Veja como se inscrever neles.

1. Faça logon no [!UICONTROL Web Personalization]. Em seu nome de logon, clique em **[!UICONTROL Configurações de Usuário]**.

   ![](assets/image2014-9-17-20-3a48-3a28.png)

1. Selecione o relatório que deseja assinar e ele é uma frequência. Clique em **[!UICONTROL Salvar]**.

   ![](assets/email-settings.png)

   Para cancelar a inscrição no relatório, desmarque e clique em **[!UICONTROL Salvar]**.
