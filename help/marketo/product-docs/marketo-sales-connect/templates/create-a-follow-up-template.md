# Criar um modelo de acompanhamento {#create-a-follow-up-template}

Com nosso recurso Campanha, ao criar um novo modelo, basta adicionar o campo dinâmico &quot;last_email_body&quot; à parte inferior do modelo. Isso automaticamente encadeará os e-mails juntos na campanha.
