# Exibir Lista de Modelos como Outro Usuário {#view-template-list-as-another-user}

Como administrador, você pode visualizar os modelos como qualquer usuário.

>[!NOTE]
>
>**Permissões de administrador necessárias**

1. Clique em **Modelos**.

   ![](assets/one.png)

1. Clique no menu suspenso **Exibir como** e selecione o usuário desejado.

   ![](assets/two.png)

1. Agora você está visualizando modelos como o usuário selecionado.

   ![](assets/three.png)

   >[!NOTE]
   >
   >Você também pode usar filtros ou a função de pesquisa junto com Exibir como para exibir o que é mais relevante para você.
