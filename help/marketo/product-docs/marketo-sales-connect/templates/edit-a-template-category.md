# Editar uma categoria de modelo {#edit-a-template-category}

Não é possível excluir uma categoria inteira ou mover modelos em massa para uma nova categoria no momento.

Você pode editar cada modelo dentro da categoria com um nome de categoria existente ou com um nome totalmente novo. Isso moverá todos os seus modelos existentes para uma pasta diferente. Quando a pasta estiver vazia, ela desaparecerá.

Estamos trabalhando para adicionar a capacidade de fazer isso em massa.
