# Editar uma Categoria de modelo {#edit-a-template-category}

Não é possível excluir uma categoria inteira ou mover modelos em massa para uma nova categoria no momento.

O que você pode fazer é editar cada modelo dentro da Categoria para um nome de Categoria existente ou para um nome novo. Isso fará com que todos os modelos existentes sejam movidos para uma pasta diferente. Quando a pasta estiver vazia, ela desaparecerá.

Estamos trabalhando para adicionar a habilidade de fazer isso em massa.
