# Fazer referência a si mesmo(a) em um campo dinâmico {#reference-yourself-in-a-dynamic-field}

Campos dinâmicos são nossa forma de mala direta.

Permitimos que você personalize seus modelos de email com atributos predefinidos como `{{my_name}}` ou `{{my_title}}`. Esses campos permitem que você faça referência a si mesmo rapidamente nos modelos de email.

Para adicionar um campo dinâmico autorreferenciável a um modelo, basta selecionar aquele que deseja usar no botão **[!UICONTROL Campos Dinâmicos do MSE]** no editor de modelo.
