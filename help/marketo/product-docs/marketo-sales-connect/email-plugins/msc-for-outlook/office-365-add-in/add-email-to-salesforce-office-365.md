# Adicionar Email a [!DNL Salesforce] (Office 365) {#add-email-to-salesforce-office}

Adicionar Email ao [!DNL Salesforce] permite que você envie por push todos os emails recebidos no [!DNL Outlook] para sua conta do [!DNL Salesforce]. Fazemos a pesquisa com base no remetente do email, portanto, no momento, só podemos usar esse botão para emails recebidos, não para emails enviados.

Estamos trabalhando para alterar a funcionalidade para permitir que você registre emails já enviados. Enquanto isso, se um email não registrar automaticamente quando enviado, ou não aparecer em suas atividades com falha no log de atividades, você poderá encaminhá-lo para seu endereço de email CCO [!DNL Salesforce].
