# Importar uma ID do Salesforce para o Sales Connect {#import-a-salesforce-id-into-sales-connect}

O Sales Connect usa o Contato ou a ID do cliente potencial (também conhecida como ID do Salesforce) para ajudar a registrar informações corretamente no Salesforce. Há algumas maneiras de anexar sua ID do Salesforce aos seus contatos no Sales Connect.

[Encaminhar para Conexão de Vendas](/help/marketo/product-docs/marketo-sales-connect/crm/salesforce-customization/push-to-sales-connect.md): Usar o Push para Conexão de Vendas criará um contato no Conexão de Vendas com todas as informações de contato principais, incluindo a ID do Salesforce do cliente potencial/contato.

[Enviar email de vendas da Marketo, Adicionar ao Campaign e Clicar para chamar botões no SFDC](/help/marketo/product-docs/marketo-sales-connect/crm/salesforce-customization/how-to-install-sales-connect-buttons-in-salesforce.md): Quando você usar qualquer um desses três botões no Salesforce, enviaremos o contato para o Sales Connect com a ID do Salesforce.

[Importar CSV](/help/marketo/product-docs/marketo-sales-connect/people/managing-contacts/import-contacts-via-csv.md): Se exportar uma lista de leads/contatos do Salesforce e incluir uma coluna com a ID do Salesforce, você poderá mapear o campo para Conexão de Vendas na importação.
