# Documento de visão geral do MCP {#overview}

TEXTO DE ATHER:

O servidor MCP público para Marketo Engage é um serviço acessível em nuvem que expõe todo o potencial das APIs REST públicas do Marketo Engage como ferramentas estruturadas e detectáveis para agentes de IA. Baseado no Protocolo de contexto de modelo (MCP), ele transforma os recursos de automação de marketing do Marketo em uma interface padronizada e controlada por esquema que os sistemas de IA podem usar de forma segura e inteligente.

Em vez de integrar manualmente os endpoints do Marketo em cada aplicativo, o servidor MCP atua como um gateway único e nativo, permitindo que os agentes compatíveis com MCP descubram as operações disponíveis, entendam os parâmetros necessários e executem ações por meio de um protocolo consistente.

Há uma série de ferramentas disponíveis que abrangem APIs REST públicas e APIs internas, que executam funções principais do Marketo como na interface do usuário, mas por meio de chamadas de API. Mais ferramentas serão adicionadas continuamente à medida que forem sendo desenvolvidas.

O servidor MCP é oferecido como um autoatendimento e pode ser configurado por meio da estrutura do Adobe IO, criando um endpoint e autenticando a conexão. Para obter mais informações e detalhes sobre a configuração, visite o site da Adobe Developer em (adicione o link aqui).

As Ferramentas do Campaign exibidas como APIs do SOAP foram convertidas em APIs REST e adicionadas.

Após a configuração, para exibir uma lista de Ferramentas disponíveis, siga estas instruções aqui (hiperlink).

NOTA: as ferramentas que não estão disponíveis são &quot;Desativar campanha&quot; e &quot;Excluir lista estática&quot; (a ser definida).
