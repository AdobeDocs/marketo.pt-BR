# Documento de visão geral do MCP {#overview}

TEXTO DE ATHER:

O Marketo Engage public MCP Server é um serviço na nuvem que torna as APIs REST do Marketo acessíveis aos agentes de IA como ferramentas estruturadas. Baseado no protocolo de contexto de modelo (MCP), ele fornece uma interface padronizada orientada por esquema que permite que os sistemas de IA interajam com o Marketo de forma segura e eficiente.

Em vez de integrar manualmente endpoints do Marketo em cada aplicativo, o servidor MCP fornece um gateway único e nativo de IA, permitindo que agentes compatíveis com MCP descubram as operações disponíveis, entendam os parâmetros necessários e executem ações por meio de um protocolo consistente.

Há uma série de ferramentas disponíveis que abrangem APIs REST públicas e APIs internas, que executam funções principais do Marketo como na interface do usuário, mas por meio de chamadas de API. Mais ferramentas serão adicionadas continuamente à medida que forem sendo desenvolvidas. &lt;— NECESSIDADE DE ESCLARECIMENTO

O servidor MCP é oferecido como autoatendimento e pode ser configurado por meio da estrutura de E/S da Adobe, criando um endpoint e autenticando a conexão. Para obter mais informações e detalhes sobre a configuração, visite o site da Adobe Developer (adicione o link aqui). ESTE? https://developer.adobe.com/marketo-apis/

As Ferramentas do Campaign exibidas como APIs do SOAP foram convertidas em APIs REST e adicionadas.

Após a configuração, para exibir uma lista de Ferramentas disponíveis, siga estas instruções aqui (hiperlink).

NOTA: as ferramentas que não estão disponíveis são &quot;Desativar campanha&quot; e &quot;Excluir lista estática&quot; (a ser definida).
