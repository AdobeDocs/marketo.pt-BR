# Configurar o botão Dismiss e Aprovar a mensagem {#set-up-the-dismiss-button-and-approve-the-message}

## Configurar propriedades do botão Dispensar  {#configure-dismiss-button-properties}

O botão Dismiss tem muitas opções para permitir configurá-lo da forma que desejar.

1. No editor, clique no botão Dismiss .

   ![](assets/image2016-5-9-10-3a23-3a37.png)

1. Se desejar alterar a cor do botão, clique no quadrado de seleção de cores. Selecione uma cor clicando nela ou inserindo os números Hex ou RGB no seletor de cores. Preto é o padrão.

   ![](assets/image2016-5-9-10-3a33-3a17.png)

1. Selecione um design de botão no menu suspenso . Os botões Arredondar incluem uma cor completa e uma opção de gradiente.

   ![](assets/image2016-5-9-10-3a35-3a46.png)

   >[!CAUTION]
   >
   >A cor do botão pode ser exibida como um X branco em um plano de fundo branco ao selecionar um design diferente do menu suspenso. Nesse caso, basta selecionar preto ou outra cor no quadrado de seleção de cores para tornar o X branco visível.

1. Você pode clicar no botão do canto esquerdo para mover o botão de rejeição para a esquerda (o lado direito é o padrão).

   ![](assets/image2016-5-9-10-3a39-3a5.png)

1. Clique na lista suspensa de cada plataforma e selecione uma ação de toque para o botão Dispensar .

   ![](assets/image2016-5-9-10-3a43-3a54.png)

   >[!NOTE]
   >
   >Você deve dar ao botão Dispensar uma ação de toque, de modo que não haja uma caixa de seleção para habilitá-lo. Dismiss Message é a opção padrão (e óbvia).

## Envolva-o {#wrap-it-up}

Todas as suas seleções para gráficos, texto e botões foram salvas automaticamente. Agora, você está pronto para terminar o trabalho.

1. Para visualizar a mensagem no aplicativo, clique no link **Ações da mensagem** e selecione **Visualizar**.

   ![](assets/image2016-5-9-10-3a58-3a38.png)

1. Visualize a mensagem no aplicativo no telefone ou tablet para verificar se ela é exibida corretamente.

   ![](assets/image2016-5-9-11-3a2-3a13.png)

1. Se estiver satisfeito com a mensagem no aplicativo, clique em **Aprovar e fechar**.

   ![](assets/image2016-5-9-11-3a8-3a52.png)

   >[!NOTE]
   >
   >Você também pode selecionar **Aprovar e fechar** diretamente do menu suspenso Ações de mensagem (consulte a etapa 1), mas por que não visualizar a mensagem primeiro, apenas para ser seguro?

1. Para fechar o editor sem aprovar, basta fechar a guia . Ele é salvo automaticamente, para que você possa retorná-lo e aprová-lo posteriormente.

   ![](assets/image2016-5-9-11-3a9-3a46.png)

Tantas escolhas, mas agora você tem uma ótima mensagem no aplicativo, pronta para ser enviada!

Agora é hora de [enviar a mensagem](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/send-your-in-app-message.md).
